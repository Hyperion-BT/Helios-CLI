# Helios-CLI

A CLI tool for compiling [Helios](https://github.com/Hyperion-BT/Helios) smart contracts.

## Installation

Dependencies:
  * node
  * npm

```shell
$ sudo npm i -g @hyperionbt/helios-cli
```

## Usage

### Compiling

```shell
$ helios compile my_script.hl --optimize --output my_script.json
```
